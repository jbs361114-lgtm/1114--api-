# 🌤 한국 단기예보 웹페이지

- **학번**: 1114  
- **이름**: 이상건  

---

## 프로젝트 소개

기상청 단기예보 조회서비스 API를 활용하여 한국의 시간별 날씨 예보를 보여주는 웹페이지입니다.

---

## 파일 구조

```
├── index.html   # 메인 HTML 페이지
├── style.css    # 스타일시트
├── app.js       # 기상청 API 호출 및 UI 로직
└── README.md    # 프로젝트 설명
```

---

## 사용 API

- **서비스명**: 기상청 단기예보 조회서비스  
- **엔드포인트**: `https://apis.data.go.kr/1360000/VilageFcstInfoService_2.0/getVilageFcst`  
- **데이터 타입**: JSON  

### 주요 예보 카테고리

| 코드 | 설명 |
|------|------|
| TMP  | 기온 (°C) |
| SKY  | 하늘 상태 (1=맑음, 3=구름많음, 4=흐림) |
| PTY  | 강수형태 (0=없음, 1=비, 2=비/눈, 3=눈, 4=소나기) |
| REH  | 습도 (%) |
| WSD  | 풍속 (m/s) |
| POP  | 강수확률 (%) |

---

## 주요 기능

1. **지역 선택**: 서울, 인천, 대전, 대구, 부산, 광주 등 10개 지역
2. **현재 날씨 요약**: 기온, 하늘 상태, 습도, 풍속, 강수확률
3. **시간별 예보**: 최대 24시간치 시간별 예보 카드 표시
4. **자동 조회**: 페이지 로드 시 서울 기준 자동 조회

## 실행 방법

VS Code에서 **Live Server** 확장을 설치한 후, `index.html`을 열고 **Go Live** 버튼 클릭.

또는 터미널에서:
```bash
npx serve .
```
