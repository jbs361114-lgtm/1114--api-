# 전주시 분리수거함 위치 정보 서비스

본 프로젝트는 전주시 오픈 API를 활용하여 제작된 위치 정보 시각화 웹사이트입니다.

## 1. 제작 정보
- **제작자**: 1114 이상건
- **학번**: 1114

## 2. 파일 구성
- `index.html`: 웹사이트의 메인 구조
- `style.css`: 상단 헤더 디자인 및 지도 레이아웃
- `app.js`: 전주시 API 연동 및 지도 마커 렌더링 로직
- `README.md`: 프로젝트 설명서

## 3. 기능 설명
- 상단 헤더에 제작자(1114 이상건) 정보 고정 노출
- Leaflet 라이브러리를 활용한 인터랙티브 지도 제공
- 전주시 분리수거함 API를 실시간으로 호출하여 마커 표시
- API 호출 실패 시 사용자 경험을 위해 샘플 데이터 로드 기능 포함
